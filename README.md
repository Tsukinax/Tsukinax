## Experience

### Founder of KlipOneAI
**2025 - Present**

Building KlipOneAI as an automation edit platform for creators, combining AI-assisted media workflows, product design, frontend engineering, backend systems, and production-focused software delivery.

- Designed the product direction for KlipOneAI and shaped the core automation edit workflow
- Built the application foundation with React, Next.js, TypeScript, Node.js, and API-driven architecture
- Developed AI-assisted media flow concepts for captions, B-roll, video editing, and creator productivity
